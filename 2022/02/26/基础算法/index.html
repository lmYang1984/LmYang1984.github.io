快速排序

算法思路：

快排属于分治算法，分治算法都有三步：

  1. 分成子问题
  2. 递归处理子问题
  3. 子问题合并

接下来证明一次划分后有:q[l..j] <= x && x <= q[j+1..r]

  • 初始条件
    两个区间均为空值，等式必然成立
  • 保持
    循环过程中开始前有 q[l..i] <= x, q[j..r] >= x

所以，i和j更新之后，下一次循环开始之前，循环不变式依然成立

这里注意自己用while时候的错误
由于使用do-while循环,所以i和j一定会!!!自增!!!,使得循环会继续下去,但是如果采用while循环(i和j的初始化做出对应的变更),i和j在特殊情况下不自增的话,循环就会卡死

例如:

  • 终止
    循环结束时，i >= j

正常情况下，按照循环不变式，我们应该会觉得结果已经显然了

因为i >= j，q[l..i] <= x, q[j..r] >= x

所以按照j来划分的话，q[l..j] <= x, q[j+1..r] >= x是显然的

可是，最后一轮循环有点特殊，因为最后一轮循环的if语句一定不会执行,所以我们只能保证如下
i >= j
q[l..i-1] <= x和q[i] >= x
q[j+1..r] >= x和q[j] <= x
由于q[l..j-1] <= x且q[j] <= x可以推出q[l..j]<=x
由此问题等证
边界情况分析之后补上
